# crud-application

A Simple CRUD Application

## Swagger Docs:

- [Docs](https://crud-application-production.up.railway.app/api-docs/)

## Built with:

- NodeJS
- Express
- Mongoose

## Running this project locally

1. clone this project
2. Add .env file and add these variables

```
PORT = 4000
MONGO_URL = your_mongoDB_url
```
